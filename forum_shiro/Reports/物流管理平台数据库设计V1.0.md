# DATABASE DESIGN

## user 用户

存储用户信息

| Field    | FieldName | Type    | Bind | Remark     |
| -------- | --------- | ------- | ---- | ---------- |
| id       | id        | bigint  | PK   |            |
| 用户名   | uername   | varchar | FK   |            |
| 密码     | password  | varchar |      |            |
| 性别     | gender    | int     |      | 1=女，0=男 |
| 联系电话 | telephone | varchar |      |            |
| 居住地   | residence | varchar |      |            |



## goods 货物

存储提交订单时对应的货物信息

| Field | FieldName | Type    | Bind | Remark                                       |

| id    | id        | bigint  | PK   |                                              |

| 种类  | type      | int     |      | enum约束:日用品,食品,文件,衣物,数码产品,其他 |
| 体积  | volume    | double  |      |                                              |
| 重量  | weight    | double  |      |                                              |
| 价值  | value     | double  |      |                                              |
